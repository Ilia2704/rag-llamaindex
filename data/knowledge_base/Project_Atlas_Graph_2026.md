# Проект Atlas: графовая навигация по корпоративным знаниям

Проект Atlas строит Graph RAG для корпоративных документов РобоТех. Цель проекта: связать документы, сотрудников, команды, политики и технические системы в одном property graph.

Мария Соколова является владельцем бизнес-требований проекта Atlas. Иван Петров является техническим лидом проекта Atlas. Анна Ким отвечает за бюджет и закупки.

Проект Atlas использует LlamaIndex PropertyGraphIndex для извлечения сущностей и отношений из документов. Граф хранится в Neo4j. Векторные чанки и hybrid search хранятся в Qdrant.

Команда AI Platform отвечает за RAG pipeline. Команда Security отвечает за требования ZTA-17. Команда PeopleOps отвечает за политику HR-REMOTE-2026.

Ключевые связи:
- Мария Соколова owns HR-REMOTE-2026.
- Иван Петров leads Project Atlas.
- Project Atlas uses Neo4j.
- Project Atlas uses Qdrant.
- Project Atlas follows ZTA-17.
