 **Заказ такси за счет компании**

**1. Основные требования**

Чтобы организовать заказ такси на счет компании, сотрудник должен удовлетворять следующим критериям:
- иметь должность, предусмотренную Положением о структуре управления;
- находиться в рабочем статусе (не отпуск, не выходной);
- иметь соответствующую финансовую аудиторию для утверждения расходов.

**2. Процедура заказа такси**

1. Сотрудник должен заполнить электронную форму, которая доступна в корпоративной сети компании.
2. После проверки и утверждения руководителем или финансовым отделом, сотрудник получает код для подтверждения заказа.
3. Код передается водителю такси через специальную беседовую систему (например, Slack).
4. Водитель проверяет код и прибывает на указанное место в установленное время.

**3. Оплата за такси**

Оплата за каждый заказ происходит автоматически из расчетного счета соответствующего отдела компании. Сотрудник не несет никаких расходов лично, а получает подтверждение о выполненном заказе по электронной почте.

**4. Заявки на регулярные заказы такси**

Для организации регулярных заказов такси (например, для перевозки клиентов) необходимо сначала обратиться в финансовый отдел с заявкой. Формат и требования к заявке устанавливаются конкретно для каждого отдела.
