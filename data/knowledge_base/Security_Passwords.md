 # Информационная безопасность: Политика Паролей

## Введение

В рамках продолжающегося усилия по сохранению конфиденциальности и интегритности систем и данных корпорации РобоТех, предоставляется эта политика паролей для обеспечения безопасного использования учетных записей и систем.

## Ключевые факты

1. Минимальная длина пароля составляет 16 символов, включая как минимум один специальный символ (например, !@#$%^&*()_+-=[]{}|;:'\",.<>?).

2. Срок действия паролей составляет 90 дней, после чего обновление пароля будет требоваться для всех пользователей.

3. Повторное использование паролей не допускается, с целью обеспечения безопасности и защиты от внешних угроз.

## Заключение

Соблюдение этой политики паролей является обязательным для всех сотрудников корпорации РобоТех, обеспечивая надежную защиту информации и систем от возможных угроз. Несоответствие этим требованиям может привести к нарушению конфиденциальности данных и потенциальным последствиям для корпорации РобоТех.

* Список допустимых специальных символов: !@#$%^&*()_+-=[]{}|;:'\",.<>?
* Инструкция по смене пароля в системе корпорации РобоТех (ссылка)
