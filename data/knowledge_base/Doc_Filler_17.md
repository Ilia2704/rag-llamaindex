 # Внутренний регламент: Разрешенное программное обеспечение для использования в компании РобоТех

1. **Общие положения**

В соответствии с политикой безопасности и эффективной работы в компании РобоТех, определены требования к программному обеспечению, которое разрешено для использования сотрудниками.

2. **Обязанности сотрудников**

Каждый сотрудник обязан использовать только программное обеспечение, которое включено в список, одобренный ИТ-департаментом компании РобоТех.

3. **Список разрешенного программного обеспечения**

Следующие приложения и системы являются разрешенными для использования сотрудниками компании РобоТех:

- Microsoft Office 365 (Word, Excel, PowerPoint, Outlook)
- Google Workspace (Docs, Sheets, Slides, Gmail)
- Slack
- Zoom
- Adobe Creative Cloud (Photoshop, Illustrator, InDesign, Acrobat Pro)
- Jira
- Confluence
- Trello

4. **Дополнительные требования**

Кроме указанного программного обеспечения, сотрудники могут использовать дополнительное ПО при получении соответствующего разрешения от ИТ-департамента. Необходимо поддерживать обновления и используйте лицензионные версии программного обеспечения для соблюдения законов о авторских правах.

5. **Наказание за нарушения**

Несоблюдение требований данного регламента может привести к дисциплинарным действиям, включая увольнение сотрудника.
