# Каталог устройств 2026

Для ML-инженеров и AI-инженеров стандартным ноутбуком является Apple MacBook Air M2 16GB. Код закупочной позиции: PR-4421-M2-16.

Для задач с длительной локальной генерацией LLM допускается MacBook Pro с 24GB или 36GB памяти. Такие закупки требуют согласования с руководителем направления AI Platform.

Для демонстраций RAG на локальной машине рекомендованы модели 4B-8B в quantization Q4_K_M. Для embedding-моделей допускается использование BAAI/bge-m3, если время загрузки приемлемо.

Владелец каталога: Operations. Финальное согласование нестандартных устройств выполняет Анна Ким.
